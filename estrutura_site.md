# Estrutura do Site Estático Nextt

## Arquivos Principais

### HTML
- `index.html` - Página inicial
- `sobre.html` - Página Sobre Nós
- `servicos.html` - Página de Serviços
- `portfolio.html` - Página de Portfólio
- `blog.html` - Página de Blog
- `contato.html` - Página de Contato

### CSS
- `css/style.css` - Estilos principais (convertidos do tema WordPress)

### JavaScript
- `js/main.js` - Scripts principais para interatividade
- `js/slider.js` - Script para o slider de depoimentos

### Imagens
- `images/logo.png` - Logo da Nextt
- `images/hero-bg.jpg` - Background da seção hero
- `images/team-*.jpg` - Fotos da equipe
- `images/portfolio-*.jpg` - Imagens do portfólio
- `images/blog-*.jpg` - Imagens do blog

## Estrutura de Navegação

### Header
- Logo Nextt (vermelho e preto)
- Menu de navegação principal
- Botão CTA "Orçamento"
- Menu mobile para dispositivos pequenos

### Footer
- Logo Nextt
- Informações de contato
- Links rápidos
- Redes sociais
- Copyright
